## Local development

```bash
# Clone Repo
$ git clone `https://github.com/HydroBlockchain/aegir-wallet`

# Install dependencies
$ yarn install

# Run project on connected Android device or running Android simulator
$ yarn android

# In a separate console run metro
$ yarn start
```
